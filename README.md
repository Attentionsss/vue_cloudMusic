Vue全家桶 + 魔改ElementUI 仿网易云客户端 (PC) 在线播放器 ☁️🎵
===

### 特性 ###
* Vue全家桶开发
* 支持歌词滚动
* 支持电台播放
* 数据持久化 (刷新页面数据不会丢失)
* 高颜值在线播放器 
* 音乐标题自适应
* 支持网易云手机号登录
* 支持评论功能
* 每日推荐歌曲
* 支持歌曲、 歌单、视频、 MV、 专辑、 电台收藏以及取消收藏
* 视频、 Mv播放


### 技术栈 ###
* ***Vue*** 全家桶
* ***ElementUI*** 魔改样式 ，体验接近网易云PC客户端
* ***vuex-persistedstate*** （Vuex持久化插件）
* ***Less***（CSS 预处理语言）
* ***better-scroll*** 歌词滚动
* ***vue-lazyload*** 图片懒加载 , 加载动画

### 后端接口 ###
https://binaryify.github.io/NeteaseCloudMusicApi

### 接口调试工具 ###

https://www.apifox.cn/
